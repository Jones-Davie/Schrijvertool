Welkom bij de demo-versie van de Schrijvertool.
Een tool wiens uiteindelijke functie is: een op maat gemaakte wikipedia om alles bij te houden in het verhaal dat ik schrijf.
Alle informatie die weergegeven wordt zal compleet dynamisch worden en bijgewerkt worden met elke nieuwe toevoeging in de database.
Dit is een verminderde versie van wat ik momenteel werkende heb. Voor de actuele code en voortgang kan je kijken op. github.com/Jones-Davie
Bij de menuoptie: Wezens. Is een voorbeeldversie te zien van een van de hoofdfuncties van de tool, waarbij alles wordt geladen uit een 'faux' database
